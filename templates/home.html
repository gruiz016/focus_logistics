This is a capstone project developed by Giovanni Ruiz. Although this is for educational purposes, this Flask application was set out to address real world problems. Helping small/startup companies manage fleets across the country.

We address companies/logistics managers needs to move thier tracking methods from using excel spreadsheets to a web a platform. The benefits are better visibility, automated distance calculations, and built in data analytics.
